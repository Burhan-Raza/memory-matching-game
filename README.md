Memory Matching Game

This project is a simple web-based Memory Matching Game where players flip cards and try to find matching pairs. The goal of the game is to match all pairs of cards by remembering their positions.

The project was developed as part of an academic activity under the supervision of the course coordinator Miss Junniah Batool.

Project Overview

The Memory Matching Game displays multiple cards placed face down. The player clicks on two cards to reveal them. If the cards match, they remain open. If they do not match, they flip back after a short time. The player continues until all pairs are successfully matched.

This project focuses on building a simple interactive game using jQuery with HTML and CSS.
